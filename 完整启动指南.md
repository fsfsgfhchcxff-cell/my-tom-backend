# 🎀 汤姆猫学习版 - 完整启动指南

## 🌟 项目结构

```
tom-learning-app/
├── tom-learning-app/          # Spring Boot 后端
│   ├── src/
│   ├── pom.xml
│   └── 启动.bat
└── frontend/                  # React 前端（新）
    ├── src/
    ├── package.json
    └── 启动前端.bat
```

## 🚀 完整启动步骤

### 第一步：启动后端 (Spring Boot)

**方式1：使用启动脚本**
```bash
双击 tom-learning-app/启动.bat
```

**方式2：使用命令行**
```bash
cd tom-learning-app
mvnw spring-boot:run
```

✅ **后端启动成功标志**：
- 看到 "Started TomLearningAppApplication"
- 端口：http://localhost:8081

---

### 第二步：启动前端 (React)

**打开新的命令行窗口**

**方式1：使用启动脚本（推荐）**
```bash
双击 frontend/启动前端.bat
```

**方式2：使用命令行**
```bash
cd frontend
npm install     # 首次运行需要安装依赖
npm start
```

✅ **前端启动成功标志**：
- 自动打开浏览器
- 地址：http://localhost:3000
- 看到可爱的登录界面 🐱

## 🎮 功能说明

### 1. 登录/注册
- 输入用户名
- 点击"开始玩耍"登录
- 或点击"创建新账号"注册

### 2. 主页 🏠
- 查看钻石余额 💎

- 浏览食物、衣服、家具
- 用钻石购买商品
- 查看钻石余额
- 购买成功有特效

### 5. 背包 🎒
- 查看已购买物品
- 装备衣服和家具
- 使用食物（会被消耗）
- 查看装备状态

## ⚠️ 常见问题

### Q: npm install 失败？
**A:** 检查 Node.js 版本，推荐使用 Node 16+
```bash
node --version
npm --version
```

### Q: 端口被占用？
**A:** 修改端口
```bash
# 后端端口（application.properties）
server.port=8082

# 前端会自动检测并使用其他端口
```

### Q: 看不到数据？
**A:** 确保后端先启动，前端后启动

### Q: 样式不生效？
**A:** 清除缓存并重新构建
```bash
cd frontend
npm run build
```
